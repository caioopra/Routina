# 📋 Planejamento Semanal

Visualizador de rotina semanal feito com Vite + React.

## Setup

```bash
npm install
npm run dev
```

Abra `http://localhost:5173` e fixe a aba no navegador.

## Editando a rotina

Edite somente o arquivo `src/rotina.json`. O hot reload do Vite atualiza o browser automaticamente.

### Com Claude Code

Dentro da pasta do projeto, basta pedir em linguagem natural:

```
"troca o bloco de academia de terça pra 18:00-19:00"
"adiciona um bloco de pesquisa na quarta à tarde"
"muda a meta do mês pra entregar o capítulo 2"
"remove o bloco de slides de segunda"
```

O `CLAUDE.md` dá todo o contexto necessário pro Claude Code entender a estrutura.
